字体 -- font-family

字体栈 --> 一组字体（使用第一个有效字体，防止前部字体缺失）

通常给font-family字体栈的最后一项指定一个通用字体类(如serif, sans-serif)

常见通用字体类

  • serif
  • sans-serif
  • monospace
  • cursive
  • fantasy

字体大小 -- font-size

绝对字体大小

像素、派卡(pica)、英寸

与祖先元素的字体大小无关

相对字体大小

百分比、em、rem(根元素字体大小)

相对于最近的“被设定过字体大小的(不一定要绝对大小)”祖先元素(一般为body)来确定


this is very important!


字体样式 -- font-style

值: italic oblique normal

设定斜体还是正体，italic和oblique效果相同

这段文字用来说明，normal可以取消所有特殊样式

一般normal用来有选择地覆盖某个默认或你设定的全局属性


字体粗细 -- font-weight

可能的值

  • 100, 200......900
  • lighter, normal, bold, bolder

最好只用bold和normal这两个值

this is bolded text with a piece of non-bolded text


字体变化 -- font-variant

值：small caps(所有小写英文字母变成小型大写字母), normal
