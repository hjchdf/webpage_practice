position属性4个值
  • static(默认)
  • relative
  • absolute
  • fixed


1.静态定位，每个元素处在常规文档流中

2.相对定位，相对于元素在文档流中的常规位置重新定位，
top, right, bottom, left属性

3.绝对定位,把元素彻底从文档流中拿出
绝对定位元素的任何祖先元素够可以成为它的定位上下文，
只要把相应祖先元素的position设定为relative即可
默认定位上下文时body

4.固定定位的定位上下文是视口，不会随页面滚动而移动
